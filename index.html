    NUESTOS Servicios

    Amplia gama de soluciones tecnológicas diseñadas para cubrir todas tus necesidades informáticas.

    Webmail ZIMBRA

    Sistema colaborativo que permite la gestión de correo electrónico, calendario, notas, etc. Este programa de software libre se usa como herramienta de comunicación, tanto interna como externa.

    GLPI

    Sistema OPEN SOURCE que permite administrar inventarios de computadoras, periféricos, equipos de red, y software de una empresa.

    Carpetas compartidas

    Mediante un servidor centralizado, puede alojar sus archivos y trabajarlos desde su equipo como una unidad de RED.

    JITSI MEET

    Mediante el sistema JITSI, plataforma de comunicación en tiempo real de código abierto, permite realizar videollamadas, conferencias web, mensajería instantánea y compartir pantalla de forma segura.

    Servicio VPN

    Mediante OPENVPN, solución de software de código abierto, le proporciona una red privada virtual (VPN) segura y confiable para poder acceder a la empresa desde cualquier lugar del mundo.

    Nube Owncloud

    Plataforma de almacenamiento y sincronización de archivos de código abierto que permite a los usuarios almacenar, organizar y compartir archivos y documentos en la nube de manera segura.
